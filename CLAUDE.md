# Claude Agents Configuration

This project utilizes a multi-agent structure. Use the `/agent <name>` command or refer to the specific agent profiles in `.claude/agents/` when context shifting:

- **Architect (`.claude/agents/architect.md`)**: High-level planning, designing the app and choosing technologies.
- **Backend Lead (`.claude/agents/backend-lead.md`)**: API design and business logic.
- **Frontend Lead (`.claude/agents/frontend-lead.md`)**: UI/UX implementation, styling (Tailwind/CSS), and state management.
- **QA Tester (`.claude/agents/qa-tester.md`)**: Debugging failing builds.
